# DSA with C++ 🚀

Welcome to my Data Structures and Algorithms (DSA) learning repository! This project tracks my journey as I master problem-solving using C++.

---

## 📌 Topics Covered

### 1. Functions
Basic to advanced concepts using functions in C++ to write clean and modular code.
* `cube.cpp` - Program to find the cube of a number.
* `factorial.cpp` - Finding the factorial of a number.
* `fibonacci.cpp` - Generating the Fibonacci series.
* `palindrome.cpp` - Checking if a number/string is a palindrome.
* `prime.cpp` - Checking for prime numbers.

### 2. Arrays
Basic to advanced array manipulation and searching/sorting algorithms.
* `sum_array.cpp` - Program to calculate the sum of all elements in an array.
* `Input_output.cpp` - Basic array input and output operations.
* `largest_array.cpp` - Finding the largest element in an array.
* `linear_search.cpp` - Searching for an element using Linear Search.
* `reverse_array.cpp` - Reversing the elements of an array.
* `sort_array.cpp` - Sorting an array.
